# my-simple-rpg
This is my simple rpg game, made with C++

### For run this program use:
```sh
    make all
``` 

### Or if you want more control
```sh
    make gen-obj
```

This generate `.o` files

```sh
    make compile
```

And this compile all archives

Finnally
```
    make run
```

To run

If you want you can run:
```
    make remove
```

For remove all `.o` files.
